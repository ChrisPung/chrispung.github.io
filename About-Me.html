Amazon SAA-C03 Exam Main Points

Domain Weight Key Topics
1. Design Resilient Architectures 30%
  • Designing highly available and fault-tolerant architectures
  • Implementing disaster recovery strategies
  • Understanding different storage options (S3, EBS, EFS)
  • Leveraging databases (RDS, DynamoDB, Redshift)
  • Decoupling applications using messaging and queuing services (SQS, SNS)
2. Design High-Performing Architectures 28%
  • Designing scalable and elastic architectures
  • Utilizing caching strategies (ElastiCache)
  • Optimizing application performance
  • Understanding content delivery networks (CloudFront)
  • Implementing load balancing (ELB)
3. Design Secure Applications and Architectures 24%
  • Implementing security best practices
  • Securing data at rest and in transit
  • Managing access control (IAM)
  • Understanding security groups and network ACLs
  • Utilizing security auditing and monitoring tools
4. Design Cost-Optimized Architectures 18%
  • Understanding cost optimization strategies
  • Utilizing cost-effective services
  • Implementing cost monitoring and analysis tools
  • Optimizing resource utilization
  • Leveraging pricing models (on-demand, reserved, spot)
